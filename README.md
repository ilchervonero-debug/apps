# ilchervonero Apps

Monorepo profesional para todas las aplicaciones de ilchervonero.

## Estructura

```
/ilframe      - Diseño de estructuras de acero
/chervo-web   - Sitio principal ilchervo.com
/djcu-app     - Aplicación DJCU
```

## Deployment

Cada app se deploya automáticamente en Vercel desde GitHub:
- iLFrame → ilchervo.com
- DJCU → djcu.ilchervo.com

## Desarrollo

```bash
cd [app-name]
npm install
npm run dev
```
